Steps:

1) Create the components.
2) Add the path & Components inside the app-routing.module.ts file.
3) Add router-outlet tag inside app.component.html file.
4) Add routerLink with all the require components.
5) Login operation:
6) Create form inside login.component.html file.
7) Add public router: Router to check checkUser() conditions.
8) Here we are using model driven form concept to checkUser().
9) If the conditions are satisfied the comtrol goes to "home" page with a welcome msg.
10) Login credentials will save in Session storage.
11) Otherwise else block will execute and show some msg.
12) After successfull log-in user can "logout" by Logout button logout() will call.
